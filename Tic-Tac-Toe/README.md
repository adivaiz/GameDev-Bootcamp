# 🕹️ Tic-Tac-Toe (Unity 2D, C#)

A simple, clean **Tic-Tac-Toe** built with **Unity 2D** and **C#** as part of my GameDev Bootcamp.  
Focus: readable code, small components, and a minimal UI.

---

## ✨ Features
- Local **2-player** mode (X vs O)
- Turn indicator & invalid-move guard
- Win/Draw detection (all 8 lines)
- **Reset / New Game** button
- Optional **basic AI** (random / easy) → room for Minimax upgrade
- Lightweight UI that scales to desktop & mobile

---

## 🔧 Tech Stack
- **Engine:** Unity (2D Core template)
- **Language:** C#
- **Target:** Desktop/WebGL (mobile-ready layout)
> Works on recent Unity versions (e.g., 6000.x or 2022+ LTS).

---

## 🚀 Getting Started

### Open & Play
1. Clone the parent repo and open in Unity Hub:
   - Project path: `GameDev-Bootcamp/Tic-Tac-Toe`
2. Open the scene: **`Assets/Scenes/Main.unity`**
3. Press **Play** ▶️
